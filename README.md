# Personal Portfolio

This repository contains the source code for my personal portfolio website.

Rather than following a traditional single-page layout, this portfolio is designed as a desktop-inspired operating system where visitors can explore my projects, experience, and technical background through interactive applications and draggable windows.

**Live Site:** https://haniksator.github.io

---

## Overview

The goal of this project was to create a portfolio that feels more like using an application than browsing a website. Every interface element was built to reinforce that experience while remaining clean, responsive, and easy to navigate.

The portfolio highlights my education, professional experience, technical skills, and personal projects, many of which were genuinely enjoyable to design, build, and refine.

Alongside my coursework at DePaul University, I've continued developing my software engineering skills through my internship at Aubot while also strengthening teamwork, communication, and problem-solving skills through my role at Whole Foods Market.

## Featured Projects

### FreshTrack
A full-stack household inventory management system built with Django and PostgreSQL.

Features include:

- Expiration-aware inventory tracking
- Household sharing
- Shopping lists
- Inventory forecasting
- Authentication and user management

---

### Remote-Controlled Car

A Raspberry Pi project written in Python using GPIOZero.

Features include:

- Motor control
- LED indicators
- Audio feedback
- Real-time movement commands
- Hardware integration

---

### Minesweeper FX *(Coming Soon)*

A JavaFX implementation of Minesweeper focused on clean architecture and responsive gameplay.

Features include:

- Custom board generation
- First-click safe logic
- Timer and flag counter
- Win/loss detection
- Separation between game logic and user interface
